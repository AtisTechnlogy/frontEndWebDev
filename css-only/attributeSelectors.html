Attribute Selectors

Presence Value
A[attr] = Mathes Presence of Attribute 1. A[attr = val] = Matches Exact value of String
  2. A[attr ^= val] = Matches Starting Value of Sub-String.
  3. A[attr $= val] = Matches Ending Value of Sub-String
  4. A[attr *= val] = Matches any where in the Sub-String
